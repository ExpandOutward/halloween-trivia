# Halloween Movie Trivia
I created this application as an entry in the W3 Schools Halloween Coding challenge in 2022. I took second place. 

# General Outline
My goal with the Halloween Movie Trivia game was to create a trivia game that offers a short, but unique and enjoyable experience for fans of the film or franchise. 

## Elements
- Good and Bad Endings: If a user answers 3 or more questions incorrectly, then the they will get the bad ending upon submitting their final answer. Else, they will view the good ending.
- Answer Feedabck: When answering a question correctly, the correct answer will highlight green and a sound clip will be played from the films further confirming the correct answer. If an incorrect answer is submitted then the submission will turn red and a sound bite of Dr. Loomis screaming from Halloween 6 will play.
- 1 Time Opening: When the user initially plays the game, an opening gif will play along with music. If the user selects the option to try again, the opening will not repeat.
- Replay: The user has the option to replay the trivia game, resetting the score.
